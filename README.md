# Streamlit Practice App (Class Exercise)

This repository contains a small Streamlit web application developed as a class exercise to practice building data-driven interfaces with Python. The app loads an employee dataset from a CSV file and provides a simple visualization of salary values.

## Tech Stack
- Python
- Streamlit
- Pandas
- Matplotlib
- Docker / Docker Compose
